# 🧭 요청 목적

현재 `RegionDetailPage.js`에 2020~2022년도 값이 표시되지 않는 현상이 있습니다.  
Cursor가 이전에 “쿼리는 제대로 작성되어 있다”고 설명했으나,  
실제로는 해당 연도 값이 존재하는지 **쿼리 결과를 실행하여 확인한 이력이 없습니다.**

**지금 요청의 목적은 "쿼리 구문 확인"이 아니라,  
"실제 데이터가 존재하는지 DB 조회 결과를 확인"하는 것입니다.**

---

# ✅ 실행해야 할 쿼리

다음 쿼리를 실제로 실행하여 결과를 5건 이상 출력해 주세요:

```sql
SELECT
  사업장명,
  [2020], [2021], [2022], [2023], [2024],
  시도,
  구군
FROM Insu_sample
WHERE 시도 = N'경기도' COLLATE Korean_CI_AS
  AND 구군 = N'화성시' COLLATE Korean_CI_AS;
```

---

# 📌 요청 의도

- 위 조건은 실제로 존재하는 구간이며,  
  데이터베이스에 [2020]~[2022] 값이 있는 사업장이 존재해야 정상입니다.
- 반환 결과를 통해 `'-'` 처리 이유가 실제 null 때문인지, 전달 누락인지 파악하려 합니다.
