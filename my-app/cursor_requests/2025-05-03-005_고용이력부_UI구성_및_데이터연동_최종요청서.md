## 2. 쿼리 정의 및 연동 구조

### 시도 단위 업체 수 집계:
```sql
SELECT 시도, COUNT(*) AS 업체수
FROM [CleanDB].CleanDB.dbo.Insu_Clean
GROUP BY 시도
ORDER BY 시도 DESC;
```

### 시도+구군 단위 업체 수 집계:
```sql
SELECT 시도, 구군, COUNT(*) AS 업체수
FROM [CleanDB].CleanDB.dbo.Insu_Clean
GROUP BY 시도, 구군
ORDER BY 시도, 구군;
```
