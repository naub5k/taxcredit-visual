# 고용이력부(가칭)

## 프로젝트 개요
본 프로젝트는 시도별/구군별 고용 현황을 시각화하여 제공하는 모바일 최적화 웹 애플리케이션입니다.

## 주요 기능
- 시도/구군별 업체 수 현황 조회
- 지역별 비율 시각화 (차트)
- 수도권(서울, 경기) 및 기타 지역 비교 분석
- 모바일 최적화 UI/UX

## 기술 스택
- React.js
- Tailwind CSS
- Recharts (데이터 시각화)

## 개발 환경 설정
```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm start

# 빌드
npm run build
```
